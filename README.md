# Simple Interactive Web Page

This project is a beginner-friendly interactive web page built with **HTML, CSS, and JavaScript**.

## Features

- Click the button to display a greeting message.
- Responsive and clean design.
- Easily customizable for learning and experimentation.

## Usage

1. Clone or download the repository.
2. Open `index.html` in your browser.

## Files

- `index.html` — Main web page structure.
- `style.css` — Styling for the page.
- `script.js` — JavaScript logic for interactivity.

## Example

The page includes:
- A title header.
- A button labeled "Say Hello".
- When the button is clicked, a greeting message appears below.

## Getting Started

You can expand this project by adding:
- More buttons for different messages.
- Forms for user input.
- Images and links.
- Interactive animations.
